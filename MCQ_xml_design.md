# Introduction #

The MCQ APP totally runs using the data from the xml. It supports both text rendering as well as html rendering.

# Details #

Here are the xml nodes which are used for describing a question to render.

<h3>Ex 1:</h3>



&lt;question&gt;


> 

&lt;htmlContent&gt;

ques1.html

&lt;/htmlContent&gt;


> 

&lt;type&gt;

radio

&lt;/type&gt;


> 

&lt;mark&gt;

2

&lt;/mark&gt;


> 

&lt;choices&gt;


> > 

&lt;choice ans="1"&gt;

A

&lt;/choice&gt;


> > 

&lt;choice&gt;

B

&lt;/choice&gt;


> > 

&lt;choice&gt;

C

&lt;/choice&gt;


> > 

&lt;choice&gt;

D

&lt;/choice&gt;



> 

&lt;/choices&gt;


> 

&lt;imagePath&gt;

&lt;/imagePath&gt;


> 

&lt;htmlExplanation&gt;

expl1.html

&lt;/htmlExplanation&gt;


> 

&lt;/question&gt;
